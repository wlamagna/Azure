## Some experiments with Azure Container Instances

* worldcup2026: an ACI with an API that returns randomly a name of a football player from the World Cup 2026 and its country.  I have more data from each player but this is the first version, if there is interest i can share more.  The ACI has an API that connects to a Cosmos DB and retrieves randomly a player and returns it.

* game1: Is a vibe coded web game of two snakes put inside a container (ACI).  This is not yet public due lack of time.

* sshone : an ssh server in a container.  A starting point for anything you can imagine: jumpbox, firewall, ids, logging, etc.  Work in progress.
